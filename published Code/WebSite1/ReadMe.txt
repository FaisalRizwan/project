This project is made in ASP.net and Sql server

before using this code make sure that visual studio10.0 ad sql server 2008 should be installed in your machine

Purpose of this project :

1. Adding a phone contact to the contacts database
2. Delete a phone contact from database
3. Modify a contact
4. search contacts
5. Add image to a contact
6. This is using web based authentication..


Note : Sql scritp is attached to this root folder as SqlScript=>script.sql
